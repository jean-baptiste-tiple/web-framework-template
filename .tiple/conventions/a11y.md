# Accessibilité

- HTML sémantique : header/nav/main/article/section/footer, un seul h1.
- alt sur toutes les images. Liens explicites (pas de "cliquez ici").
- Composants interactifs au clavier. Préférer natif (<details>, <dialog>, <button>).
- Contraste suffisant (tokens prévus pour). Focus visible conservé.
- Îlots Solid : aria-* corrects, état annoncé.
