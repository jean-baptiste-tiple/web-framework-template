# Starters

Le **socle** inclut déjà, prêts à l'emploi : blog, pages, landings, SEO/JSON-LD, sitemap, RSS, formulaire de contact (vanilla) et réglages globaux (settings/site.json). Docs de référence : `blog/`, `forms/`.

Modules **OPT-IN** (NON installés par défaut) — chaque README explique comment les ACTIVER :
- solid : îlots SolidJS pour interactivité riche (sinon préférer `<script>` vanilla).
- i18n : routing multilingue (fr racine + /en).

Édition du contenu : directement dans les fichiers .md/.mdx/.json (pas d'interface d'admin).
